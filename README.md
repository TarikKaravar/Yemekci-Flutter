## Başlangıç


### Gereksinimler

- Flutter SDK (sürüm 10.0.19045.5131)

- Dart SDK


### Kurulum Adımları

1.  Projeyi bilgisayarınıza indirin:

git  clone [https://github.com/TarikKaravar/Yemekci-Flutter.git]

2.  Proje klasörüne gidin:

cd [Yemekci-Flutter]

3.  Gerekli paketleri yükleyin:

flutter  pub  get

4.  Uygulamayı başlatın:

flutter  run
